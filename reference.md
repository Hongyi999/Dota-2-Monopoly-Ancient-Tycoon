# Website Reference

**Source URL:** https://helen.faces.site/

---

This repository contains a code recreation of the website above.
The site is a one-page creative portfolio featuring:

- Hero section with name, title, and a short tagline
- Animated stat counters
- About / bio section with skill tags
- Project showcase with marquee scroll
- Contact section
- Dark theme with a grainy texture overlay
- Scroll-triggered animations throughout
- Fully responsive layout
